# Taxi DWH Search

Командный DWH-проект, выполненный при поддержке Яндекса, был направлен на сбор и анализ сообщений из сообществ таксистов.

Моя часть проекта — реализация поиска и фильтрации сообщений в CDM-слое. Я спроектировала таблицу с текстами сообщений и метаданными для поиска, настроила полнотекстовый поиск в PostgreSQL через `tsvector` и GIN-индекс, добавила фильтры по дате, источнику, чату и автору, а также подготовила SQL-запросы для запуска через Superset.

## Что было реализовано

- CDM-таблица для поиска по сообщениям
- хранение текста сообщения, времени отправки, источника, чата и автора
- полнотекстовый поиск по русскоязычным сообщениям
- поле `search_vector` типа `tsvector`
- GIN-индекс для ускорения поиска
- фильтры по дате, источнику, чату и автору
- типовые SQL-запросы для Superset

## Технологии

SQL, PostgreSQL, Superset, Docker, Python
